## This collection consists of the following CLI tools

1. Todo List Tool

- Allows adding,removing,toggle complete status
- Includes presets like removing all completed tasks, mark all tasks as completed

2. Timer(Pomodoro) Tool

- Allows custom timer with minutes as input
- Allows a full uniterrupted pomodoro timer cycle
- Allows individual fixed sessions from the pomodoro: work and break
- Sound notification when timer ends
